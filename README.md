🚨 Metro Police System 🇧🇩

A Comprehensive Digital Policing Platform

The Metro Police System is a full-stack web application designed to modernize and digitize policing services for Bangladesh Metro Police. It bridges the gap between citizens and law enforcement by providing fast, secure, and accessible online services such as crime reporting, traffic management, and emergency response.

🌐 Overview

This platform enhances public safety and operational efficiency by integrating citizen services with intelligent law enforcement tools. It enables real-time interaction, data-driven decision-making, and streamlined workflows for police departments.

⚡ Key Features
🧑‍🤝‍🧑 Services for Citizens
📘 Online General Diary (GD)
Report crimes, lost items, or missing persons directly through the platform.
🚗 Traffic Fine Management
Check traffic violations using vehicle numbers and pay fines online (simulated).
📡 Emergency SOS System
Send instant alerts to nearby police stations with location data.
🪪 Police Clearance Certificate (PCC)
Apply for and track clearance certificates online.
🧭 Safe Route Navigation
Find safer travel routes based on crime data and patrol activity.
🤖 Cyberbullying Assistance
AI-based assistant to guide victims and provide support.
🛡️ Tools for Law Enforcement
🗃️ Criminal Database
Search and manage criminal records with simulated face recognition.
🗺️ Crime Mapping and Heatmaps
Visualize crime hotspots and police patrol density.
📈 Crime Prediction
Machine learning-based analysis of potential crime patterns.
🔎 Missing Person Auto-Match
Automatically match missing persons with unidentified individuals.
🖥️ Admin Dashboard
Manage reports, update case statuses, and monitor operations.
⚙️ Technology Stack

Frontend: React.js (Vite), Tailwind CSS
Backend: Node.js, Express.js
Database: MySQL
Authentication: JWT, Bcrypt

🧩 Getting Started

Database Setup:
Import database/schema.sql into your MySQL server (e.g., XAMPP)

Backend Setup:
Run npm install and npm start inside the backend directory

Frontend Setup:
Run npm install and npm run dev in the project root

For detailed instructions, see README_SETUP.md
